## 对 HAL 设计的关键发现

| # | 发现 | 影响 |
|---|------|------|
| 1 | 驱动采用 `Driver(Node) → Logic(non-Node) → SDK` 三层架构 | HAL 应在 Logic 层以下实现 |
| 2 | 使用 `stream_index_pair = {StreamType, Index}` 作为统一流标识 | HAL 需要等效的流键类型 |
| 3 | 100+ 可配置参数 | 需要良好的参数管理框架 |
| 4 | 12 种深度后处理滤波器，可组合配置 | 需要可配置的处理管线（IFilterChain） |
| 5 | 三平台 MJPEG 解码（CPU / Jetson / RK） | 需要解码器工厂抽象（IDecoderFactory） |
| 6 | 支持深度相机和 LiDAR 两类设备 | 需要不同的传感器输出模型 |
| 7 | 多种硬件 / 软件同步模式 | 需要独立的同步管理器（ISyncManager） |
