# 图像处理管线深度分析

## 1. 图像格式支持

### 1.1 SDK 格式（OBFormat）

| 格式 | 说明 | 典型应用 |
|------|------|---------|
| OB_FORMAT_MJPG | Motion JPEG 压缩 | USB2.0 彩色流 |
| OB_FORMAT_YUYV | YUV 4:2:2 | 彩色流 |
| OB_FORMAT_UYVY | YUV 4:2:2（另一种排列） | 彩色流 |
| OB_FORMAT_NV21 | YUV 4:2:0 半平面 | 彩色流 |
| OB_FORMAT_NV12 | YUV 4:2:0 半平面 | 彩色流 |
| OB_FORMAT_RGB  | RGB 8:8:8 | 彩色流 |
| OB_FORMAT_BGR  | BGR 8:8:8 | 彩色流 |
| OB_FORMAT_BGRA | BGRA 8:8:8:8 | 彩色流 |
| OB_FORMAT_Y16  | 16 位灰度 | 深度流、IR 流 |
| OB_FORMAT_Y8   | 8 位灰度  | IR 流 |
| OB_FORMAT_H264 | H.264 编码 | 网络传输彩色流 |
| OB_FORMAT_H265 | H.265 编码 | 网络传输彩色流 |

### 1.2 格式到 ROS 编码映射

```
OB_FORMAT_YUYV/UYVY/NV21/NV12 → FormatConvertFilter → "rgb8"
OB_FORMAT_MJPG                 → JPEGDecoder          → "bgr8"
OB_FORMAT_H264/H265            → 硬件解码器            → "bgr8"
OB_FORMAT_RGB                  → 直接使用              → "rgb8"
OB_FORMAT_BGR                  → 直接使用              → "bgr8"
OB_FORMAT_Y16                  → 直接使用              → "mono16"
OB_FORMAT_Y8                   → 直接使用              → "mono8"
```

## 2. 颜色帧解码管线

### 2.1 MJPEG 三平台解码策略

```
MJPEG Frame
    │
    ├── [Jetson 平台] USE_NV_HW_DECODER
    │   └── JetsonNVDecoder
    │       ├── NvJPEGDecoder（Jetson Multimedia API）
    │       ├── NvBufferTransform（颜色空间转换）
    │       └── 输出：BGR/RGB Mat
    │
    ├── [RK 平台] USE_RK_HW_DECODER
    │   └── RkMppDecoder
    │       ├── MPP Decoder（RK MPP API）
    │       ├── RGA 颜色转换（librga 可用时）/ libyuv
    │       └── 输出：BGR Mat
    │
    └── [通用平台] 软件解码
        └── JPEGDecoder（libjpeg-turbo）
            ├── tjDecompress2() → RGB/BGR
            └── 输出：BGR Mat
```

### 2.2 颜色帧异步处理

```
onNewFrameSetCallback()
    │
    ├── 提取 Color Frame → color_frame_queue_（生产者）
    │
    └── colorFrameThread_（独立线程，消费者）
        └── onNewColorFrameCallback()
            ├── 从队列取帧
            ├── softwareDecodeColorFrame() / FormatConvert
            ├── processColorFrameFilter()
            └── onNewFrameCallback(COLOR) → 发布
```

## 3. 深度滤波器链

### 3.1 滤波器顺序（setupDepthPostProcessFilter）

```
原始深度帧
    │
    ▼ ① DecimationFilter          降采样（scale: 2/4/8）
    ▼ ② HDRMerge                  HDR 合成（多曝光合并）
    ▼ ③ SequenceIdFilter          多曝光模式序列选择
    ▼ ④ ThresholdFilter           阈值过滤（min/max）
    ▼ ⑤ HardwareNoiseRemovalFilter 硬件噪声去除
    ▼ ⑥ NoiseRemovalFilter        软件噪声去除（min_diff=256, max_size=80）
    ▼ ⑦ SpatialFilter             空间平滑（alpha, diff, magnitude, radius）
    ▼ ⑧ SpatialFastFilter         快速空间滤波变体
    ▼ ⑨ SpatialModerateFilter     中等空间滤波变体
    ▼ ⑩ TemporalFilter            时间平滑（diff_threshold, weight）
    ▼ ⑪ HoleFillingFilter         孔洞填充（nearest/farthest 等模式）
    ▼ ⑫ DisparityToDepth          视差转深度
    │
    ▼
处理后的深度帧
```

### 3.2 滤波器状态发布

```cpp
// 以 JSON 格式发布当前滤波器配置
nlohmann::json filter_status_;
filter_status_["spatial"]["enable"] = enable_spatial_filter_;
filter_status_["spatial"]["alpha"]  = spatial_filter_alpha_;
// ...
filter_status_pub_->publish(filter_status_.dump());
```

## 6. 图像翻转 / 镜像 / 旋转

```cpp
onNewFrameCallback(frame, stream_index) {
    cv::Mat image(height, width, cv_type, frame->data());

    if (mirror_stream_[sip]) cv::flip(image, image, 1);  // 镜像
    if (flip_stream_[sip])   cv::flip(image, image, 0);  // 翻转
    if (rotation > 0)        cv::rotate(image, image, rotation_code);
}
```

## 7. 畸变矫正

```cpp
// enable_color_undistortion_ = true
UndistortedImageResult undistortImage(image, intrinsic, distortion) {
    cv::Mat K = ...; // 相机矩阵
    cv::Mat D = ...; // 畸变系数 k1,k2,p1,p2,k3
    cv::Mat new_K = cv::getOptimalNewCameraMatrix(K, D, size, 0);
    cv::undistort(image, result, K, D, new_K);

}
```
