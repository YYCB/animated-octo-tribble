# 消息与服务定义详解

## 1. 自定义消息（Messages）

### 1.1 DeviceInfo.msg

```
std_msgs/Header header
string name                        # 设备名称（如 "Gemini 335"）
string serial_number               # 序列号
string firmware_version            # 固件版本
string supported_min_sdk_version   # 支持的最低 SDK 版本
string current_sdk_version         # 当前 SDK 版本
string hardware_version            # 硬件版本
```

**用途**：通过 GetDeviceInfo Service 返回，包含设备的基本硬件信息。

---

### 1.2 DeviceStatus.msg

```
std_msgs/Header header

# 彩色流状态
float64 color_frame_rate_cur/avg/min/max   # 帧率统计（Hz）
float64 color_delay_ms_cur/avg/min/max     # 延迟统计（ms）

# 深度流状态（同上结构）
float64 depth_frame_rate_cur/avg/min/max
float64 depth_delay_ms_cur/avg/min/max

# 设备信息
bool   device_online          # 设备在线状态
string connection_type        # 连接类型（USB2.0/USB3.0/GigE）

# 标定状态
bool customer_calibration_ready    # 用户标定是否就绪
bool calibration_from_factory      # 出厂标定
bool calibration_from_launch_param # Launch 参数标定
```

**用途**：由 deviceStatusTimer 定期发布，监控设备运行状态。

---

### 1.3 Extrinsics.msg

```
std_msgs/Header header
float64[9] rotation      # 3×3 旋转矩阵（行主序）
float64[3] translation   # 3D 平移向量（米）
```

**用途**：传感器间的外参变换（深度到彩色、深度到 IR 等）。

---

### 1.4 IMUInfo.msg

```
std_msgs/Header header
float64   noise_density             # 噪声密度
float64   random_walk               # 随机游走
float64   reference_temperature     # 参考温度
float64[3] bias                     # 3 轴偏移
float64[3] gravity                  # 重力向量
float64[9] scale_misalignment       # 刻度与非正交矩阵
float64[9] temperature_slope        # 温度斜率矩阵
```

**用途**：IMU 传感器的内在校准参数，用于数据补偿。

---

### 1.5 Metadata.msg

```
std_msgs/Header header
string json_data    # JSON 格式的帧元数据
```

**用途**：帧级元数据，包含曝光时间、增益、白平衡、传感器温度等。

---

### 1.6 RGBD.msg

```
std_msgs/Header header
sensor_msgs/CameraInfo rgb_camera_info    # 彩色相机信息
sensor_msgs/CameraInfo depth_camera_info  # 深度相机信息
sensor_msgs/Image rgb                     # 彩色图像
sensor_msgs/Image depth                   # 深度图像
```

**用途**：将 RGB 和 Depth 打包在一条消息中，确保时间戳同步。

---

## 2. 自定义服务（Services）

### 2.1 基础服务类型

| 服务 | 请求 | 响应 |
|------|------|------|
| GetBool | — | `bool data` |
| GetInt32 | — | `int32 data` |
| GetString | — | `string data` |
| SetInt32 | `int32 data` | — |
| SetString | `string data` | — |

### 2.2 GetDeviceInfo.srv

```
---
orbbec_camera_msgs/DeviceInfo device_info
```

### 2.3 GetCameraInfo.srv

```
---
sensor_msgs/CameraInfo info
```

### 2.4 SetFilter.srv

```
string filter_name   # 滤波器名称（如 "spatial", "temporal"）
bool   enable        # 使能 / 禁用
int32  value         # 参数值
---
bool success
```

### 2.5 SetArrays.srv

```
int32[] data         # 整数数组（如 AE ROI: [left, top, right, bottom]）
---
bool success
```

### 2.6 GetUserCalibParams.srv / SetUserCalibParams.srv

用于读写用户自定义标定参数到设备存储。

---

## 3. 标准 ROS2 消息使用

| 标准消息 | 用途 |
|---------|------|
| sensor_msgs/Image | 图像数据传输 |
| sensor_msgs/CameraInfo | 相机内参和畸变 |
| sensor_msgs/PointCloud2 | 点云数据 |
| sensor_msgs/Imu | IMU 数据 |
| sensor_msgs/LaserScan | 2D 激光扫描 |
| geometry_msgs/TransformStamped | TF 变换 |
| std_msgs/String | 滤波器状态 JSON |
| diagnostic_msgs/* | 诊断信息 |

---

## 4. 对 HAL 的消息抽象

HAL 层定义与 ROS2 无关的数据结构，由 ROS2 接口层负责转换：

```
HAL: CameraFrame  → ROS2 接口: sensor_msgs::msg::Image
HAL: PointCloud   → ROS2 接口: sensor_msgs::msg::PointCloud2
HAL: IMUSample    → ROS2 接口: sensor_msgs::msg::Imu
HAL: DeviceInfo   → ROS2 接口: orbbec_camera_msgs::msg::DeviceInfo
HAL: Extrinsics   → ROS2 接口: orbbec_camera_msgs::msg::Extrinsics
```
