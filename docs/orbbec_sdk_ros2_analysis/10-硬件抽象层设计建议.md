# 硬件抽象层（HAL）设计建议

## 1. 设计目标

基于 OrbbecSDK_ROS2 v2.7.6 源码分析，提出硬件抽象层设计建议。目标：
- **解耦**：将 Orbbec SDK 依赖从业务逻辑中剥离
- **通用性**：支持多种深度相机品牌（Orbbec、Intel RealSense、Stereolabs ZED 等）
- **可扩展**：方便添加新设备驱动
- **ROS2 兼容**：保持与 ROS2 生态的无缝集成

---

## 2. 整体架构

```
┌──────────────────────────────────────────────────┐
│              应用层（Application Layer）           │
│   SLAM、导航、3D 重建、人体识别等                  │
├──────────────────────────────────────────────────┤
│              ROS2 接口层（ROS2 Interface）         │
│   CameraNodeDriver（统一 ROS2 节点）               │
│   - Topic 发布 / Service 处理 / TF 管理 / 参数管理 │
├──────────────────────────────────────────────────┤
│          硬件抽象层（HAL）                         │
│   IDeviceManager  IDevice  ISensor               │
│   IStreamProfile  IFrame   IFilter               │
│   IDecoder        ISyncManager                   │
├──────────────────────────────────────────────────┤
│           驱动实现层（Driver Implementations）     │
│   OrbbecDriver   RealSenseDriver   ZEDDriver      │
├──────────────────────────────────────────────────┤
│              硬件层（Hardware）                    │
└──────────────────────────────────────────────────┘
```

## 3. 核心接口设计

### 3.1 IDeviceManager

```cpp
class IDeviceManager {
public:
    virtual ~IDeviceManager() = default;

    virtual std::vector<DeviceInfo> enumerateDevices() = 0;

    virtual std::shared_ptr<IDevice> createDevice(
        const DeviceSelector& selector) = 0;

    using DeviceChangeCallback = std::function<void(
        const std::vector<DeviceInfo>& added,
        const std::vector<DeviceInfo>& removed)>;
    virtual void registerDeviceChangeCallback(DeviceChangeCallback) = 0;

    virtual void enableNetworkDeviceEnumeration(bool enable) = 0;
    virtual std::shared_ptr<IDevice> connectNetworkDevice(
        const std::string& ip, int port) = 0;
};

struct DeviceSelector {
    std::string serial_number;
    std::string usb_port;
    std::string ip_address;
    int         device_index = -1;  // -1 = 自动
};
```

### 3.2 IDevice

```cpp
class IDevice {
public:
    virtual ~IDevice() = default;

    virtual DeviceInfo getDeviceInfo()  const = 0;
    virtual DeviceType getDeviceType()  const = 0;

    virtual std::vector<SensorType>       getSupportedSensors() const = 0;
    virtual std::shared_ptr<ISensor>      getSensor(SensorType)       = 0;

    virtual void startStreams(const StreamConfig& config) = 0;
    virtual void stopStreams()                            = 0;
    virtual bool isStreaming()                     const = 0;

    using FrameSetCallback = std::function<void(std::shared_ptr<IFrameSet>)>;
    virtual void setFrameCallback(FrameSetCallback) = 0;

    // 属性控制
    virtual void  setProperty(PropertyId id, int   value) = 0;
    virtual int   getProperty(PropertyId id)        const = 0;
    virtual void  setProperty(PropertyId id, float  value) = 0;
    virtual float getPropertyFloat(PropertyId id)   const = 0;
    virtual void  setProperty(PropertyId id, bool   value) = 0;
    virtual bool  getPropertyBool(PropertyId id)    const = 0;
    virtual PropertyRange getPropertyRange(PropertyId id) const = 0;

    // 校准数据
    virtual CameraIntrinsics getIntrinsics(SensorType) const = 0;
    virtual CameraDistortion getDistortion(SensorType) const = 0;
    virtual Extrinsics       getExtrinsics(SensorType from, SensorType to) const = 0;

    // 设备控制
    virtual void reboot()        = 0;
    virtual void syncHostTime()  = 0;
    virtual void triggerCapture()= 0;

    // IMU
    virtual void startIMU(const IMUConfig& config) = 0;
    virtual void stopIMU()                         = 0;
    using IMUCallback = std::function<void(const IMUSample&)>;
    virtual void setIMUCallback(IMUCallback)       = 0;
};
```

class DriverRegistry {

    void registerDriver(const std::string& name, DriverFactory factory);

    std::vector<std::string> getAvailableDrivers() const;
};

#define REGISTER_CAMERA_DRIVER(name, cls) \
    static bool _reg_##name = (DriverRegistry::instance() \
        .registerDriver(#name, []{ return std::make_shared<cls>(); }), true)

REGISTER_CAMERA_DRIVER(orbbec, OrbbecDeviceManager);

## 6. Orbbec 驱动实现示例

```cpp
class OrbbecDeviceManager : public IDeviceManager {
    std::unique_ptr<ob::Context> ctx_;
public:
    std::vector<DeviceInfo> enumerateDevices() override {
        auto list = ctx_->queryDeviceList();
        std::vector<DeviceInfo> result;
        for (int i = 0; i < list->deviceCount(); i++)
            result.push_back(convertDeviceInfo(list->getDevice(i)->getDeviceInfo()));
        return result;
    }
    std::shared_ptr<IDevice> createDevice(const DeviceSelector& sel) override {
        return std::make_shared<OrbbecDevice>(selectObDevice(ctx_->queryDeviceList(), sel));
    }
};

class OrbbecDevice : public IDevice {
    std::shared_ptr<ob::Device> device_;
    std::unique_ptr<ob::Pipeline> pipeline_;
public:
    void setProperty(PropertyId id, int value) override {
        device_->setIntProperty(mapToObProperty(id), value);
    }
};
```
