## 6.6 地图小结

```
全球具身智能竞争格局（2025–2026 视角）

技术深度 + 长期投入：美国（Google DM / NVIDIA / Tesla / Figure / PI）
量产 + 供应链 + 性价比：中国（宇树 / 智元 / 银河 / 星动 / 傅利叶 / Optimus 国产链）
研究底蕴 + 局部突破：欧洲（ETH / NEURA）、日本（TRI / Toyota / Honda）
生态平台基石：NVIDIA、Google、Hugging Face、ROS 社区
```

未来 3 年值得重点跟踪的"风向标"：
1. Physical Intelligence、Figure / 1X 商用化进度；
2. Tesla Optimus 量产与售价；
3. NVIDIA GR00T + Cosmos 在中国 OEM 的渗透；
4. 智元、宇树、银河通用之间的产品差距演化；
5. 触觉 + 灵巧手国产玩家是否出现"全球级公司"。
