# 10 政策环境

## 10.1 中国

### 10.1.1 国家级政策

| 时间 | 主体 | 文件 / 事件 | 要点 |
|------|------|-------------|------|
| 2023.10 | 工信部 | 《人形机器人创新发展指导意见》 | 首次明确"2025 年初步建立、2027 年成熟"两阶段目标；将"大脑、小脑、肢体"作为关键技术 |
| 2024.05 | 国务院 | "人工智能 +" 行动 | 把具身智能列入战略前沿 |
| 2024 | 工信部 / 科技部 | 揭榜挂帅、专精特新 | 鼓励核心零部件、整机、模型攻关 |
| 2024.11 | 工信部 | 《人形机器人重点产品和技术目录》（征求意见稿） | 划定重点方向 |
| 2025 | 政府工作报告 | "具身智能 / 智能机器人" 进入国家级关键词 | — |
| 2025–2026 | 工信部 / 标准院 | GB / T 标准制定 | 安全、术语、测试方法体系化 |

### 10.1.2 地方政策（代表）

- **北京**：《北京市机器人产业创新发展行动方案 (2023–2025)》、亦庄人形机器人产业基地、国地具身智能创新中心；
- **上海**：《上海市推动人形机器人产业创新发展实施方案》、张江 + 浦东聚集，国地中心；
- **深圳**：《深圳市加快推进人形机器人产业创新发展的若干措施》、坪山 + 南山布局；
- **广东 / 浙江 / 江苏 / 安徽**：相继出台产业基金 + 应用场景试点政策。

### 10.1.3 国家级创新平台

- **国家地方共建具身智能机器人创新中心**（北京、上海等地分别挂牌）；
- **国家高端装备制造业创新中心**（机器人方向）；
- **智源研究院 / 上海 AI Lab / 浦江实验室 / 北京通研院 / 之江实验室** 等扮演公共算力 / 数据 / 模型节点。

### 10.1.4 中国政策方向特征

1. **"硬件 + 模型 + 应用"全链条扶持**：与新能源车类似的产业政策方法论；
2. **重视关键零部件国产化**：减速器 / 丝杠 / 电机 / 传感器列入扶持清单；
3. **场景驱动**：地方政府主动开放工厂、医院、电力、应急、养老场景作为试点；
4. **数据基础设施**：建设公共训练数据集与开源模型；
5. **标准 + 安全 + 伦理**：2025 起加速标准化。

## 10.2 美国

| 时间 | 主体 | 内容 |
|------|------|------|
| 2022 | 白宫 | AI Bill of Rights（蓝图） |
| 2023.10 | 拜登政府 | EO 14110 "Safe, Secure, and Trustworthy AI" |
| 2024 | 国会 | NDAA / DARPA 持续投入机器人与自主系统 |
| 2025 | 特朗普政府 | 撤销部分 EO，强调"AI 领先"，对中国出口管制扩大（含先进 GPU / 先进机器人） |
| 持续 | 商务部 BIS | 出口管制清单不断更新，影响中国具身公司获取 H100/H200 / Thor 等高端算力 |

特征：
- **没有专门"人形机器人法"**，但通过 AI / 出口管制 / 国防需求间接驱动；
- **军方需求显著**：DARPA、空军、陆军持续招标四足、无人机、协作机器人；
- **市场化为主**，私人资本为主导。

## 10.3 欧盟

- **EU AI Act**（2024 生效，2025–2027 分阶段执行）：高风险 AI 包括"机器人 + 人体安全"场景；
- **机械指令 (Machinery Regulation, 2023/1230)**：将"AI 系统作为安全组件"明确纳入合规要求；
- **CE 认证 + ISO 13482 (个人服务机器人)** 是欧洲市场必须；
- 资金：Horizon Europe 持续投入 AI + 机器人。

特征：监管最严，**合规成本最高**；但也提供了清晰的市场准入框架。

## 10.4 日本与韩国

### 日本
- 长期把机器人列为战略产业（NEDO 系列计划）；
- 厚生劳动省推动护理机器人国家战略；
- 现实中以"工业机器人四大家族"为支柱，新势力较少。

### 韩国
- 2023 出台 *Intelligent Robot Promotion Act* 修订；
- 三星、LG、现代（含 Boston Dynamics）布局家庭与汽车场景；
- 政府资金倾向"老龄化 + 制造业"。

## 10.5 国际标准与认证

| 标准 | 说明 |
|------|------|
| **ISO 10218 / TS 15066** | 工业机器人 + 协作机器人 |
| **ISO 13482** | 个人服务机器人安全 |
| **IEC 62443** | 工控网络安全 |
| **IEC 61508 / 26262** | 功能安全（SIL / ASIL）参照 |
| **ISO/IEC 23894** | AI 风险管理 |

| **GB / T 系列**（中国） | 2024–2026 加速制定，对标 ISO |

## 10.6 监管热点议题

1. **责任归属**：机器人造成人身伤害时的开发者 / 集成商 / 用户 / 平台责任划分；
2. **数据隐私**：家庭与医疗机器人的视音频采集；
3. **作业代替**：劳动力替代 / 转岗 / 再培训；
4. **军事化使用**：四足 + 武装、自主作战风险；
5. **儿童与老人**：陪伴 / 教育机器人特殊保护；
6. **跨境数据**：训练数据合规、模型导出、出口管制（GPU、动力学软件、特定零部件）。

## 10.7 政策影响判断

- 中国：**最积极**，全链条扶持 + 地方场景开放，推动出货量；
- 美国：**资本与算力最强**，但出口管制反向制约；
- 欧盟：**最严**，准入门槛高，可能孕育以"安全合规"为壁垒的玩家；
- 韩日：聚焦垂直市场（护理、工业）。

## 10.8 小结

具身智能正在迈入"政策、资本、技术"三轮共振的时刻：
- 政策端的最大变量是 **军事化使用** 与 **就业冲击** 引发的反制；
- 标准化进程将影响 2027 之后的市场结构（不达标无法销售/进入大客户）；
- 中国制造业基础 + 政策红利，是 2026–2030 期间国产玩家的战略窗口。
