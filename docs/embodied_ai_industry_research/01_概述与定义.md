# 01 概述与定义

## 1.1 什么是具身智能（Embodied AI）

**具身智能（Embodied AI, EAI）** 指智能体通过"具有物理身体"的形式，与真实或仿真世界进行**感知-决策-行动**闭环，从而完成认知、操作和复杂任务的智能形态。

学术定义最早由 Rodney Brooks（MIT，1990s）等机器人学家提出，强调 "intelligence requires a body"——认知能力必须建立在与环境交互的身体经验之上。在大模型时代，这一概念被重新激活并被赋予新内涵：

> 经典定义（1990s）：智能 ≠ 抽象推理，必须由"身体 + 环境"涌现。
> 当代定义（2023 后）：以**大模型作为大脑**、**机器人本体作为身体**，形成可泛化、能交互、会学习的物理智能体。

## 1.2 与相关概念的关系

```
                ┌────────────────────────────────────┐
                │        通用人工智能 AGI            │
                │   （目标：人类级别的通用智能）      │
                └────────────┬───────────────────────┘
                             │ 是物理化路径
                             ▼
   ┌──────────────────────────────────────────────┐
   │           具身智能 Embodied AI                │
   │   ┌────────────┐    ┌────────────────────┐   │
   │   │  大脑：     │    │  身体：             │   │
   │   │ LLM / VLM  │    │ 人形 / 四足 / 臂    │   │

   │   └────────────┘    └────────────────────┘   │
   └──────────────┬───────────────────┬───────────┘

        传统机器人学 Robotics    传统 AI / NLP / CV
        （控制+规划+SLAM）      （感知与认知，无身体）
```

| 维度 | 传统机器人 | 传统 AI（含 LLM） | 具身智能 |
|------|-----------|------------------|---------|
| 智能来源 | 工程化规则、控制论 | 数据驱动统计学习 | 大模型 + 物理交互经验 |
| 任务范围 | 单一、结构化 | 数字世界 | 开放、物理世界 |
| 泛化能力 | 弱 | 任务内强、跨任务弱 | 期望跨任务、跨本体泛化 |
| 数据来源 | 人工编程 | 互联网文本 / 图像 | 真机示教 + 仿真 + 网络数据 |
| 评价指标 | 精度、节拍 | benchmark 准确率 | 任务成功率 + 安全 + 通用性 |

## 1.3 发展脉络

### 1) 经典机器人时代（1960s–2010s）
- 工业机械臂（Unimate, 1961）→ ABB / FANUC / KUKA / 安川"四大家族"主导工业机器人；
- DARPA Grand Challenge（2004 起）推动自动驾驶；
- 波士顿动力（Boston Dynamics）以 Big Dog / Atlas / Spot 长期引领腿足运动控制；
- 主导技术：MPC / WBC / SLAM / 运动规划，编程式智能、泛化弱。

### 2) 数据驱动机器人学（2015–2022）
- 深度强化学习登场：DeepMind / OpenAI 在仿真中训练多种 manipulation 与 locomotion 策略；
- 模仿学习兴起：Behavioral Cloning、DAgger、Diffusion Policy；
- ROS / ROS 2 成为机器人软件事实标准；
- 仿真器（MuJoCo, Isaac Gym, Habitat）让大规模并行训练成为可能；
- 但**通用性、Sim2Real、数据效率**始终是瓶颈。

### 3) 大模型 + 具身（2022/2023 至今）
关键事件：
- **2022.04** Google SayCan：首次把 LLM 当"高层规划器"接到真实机器人；
- **2022.12** Google RT-1：示教数据驱动的 Transformer 机器人策略；
- **2023.07** Google RT-2：第一个大规模 **VLA**（视觉-语言-动作）模型；
- **2023.08** Tesla Optimus 进展演示，引爆人形机器人资本市场；
- **2024.03** Figure × OpenAI 演示家居场景对话操作；
- **2024.06** NVIDIA 发布 GR00T 人形机器人通用基础模型项目；
- **2024.10** Physical Intelligence (π0) VLA 模型发布；
- **2025** 国内"百形大战"：宇树、智元、银河通用、星尘智能、星动纪元、银河通用等密集发布人形产品与基础模型；
- **2025** 世界模型路线（V-JEPA、1X World Model、Genie 系列）成为另一主流范式。

### 4) 当前阶段（2026 视角）
- 技术上从"模块化 pipeline"向"端到端 VLA + 世界模型"演进；
- 硬件上"人形大爆发 + 灵巧手革命"；
- 商业上：**工业/商业垂直场景率先验证 PMF**，家庭场景仍处早期；
- 行业焦点从"能不能跑能不能站"转向"能不能完成有用的任务、可靠到能上岗"。

## 1.4 为什么"现在"重要

1. **大模型外溢**：LLM/VLM 提供了前所未有的语义先验和泛化能力，解决了机器人长期缺乏"常识"的痛点；
2. **硬件成本下降**：谐波减速器、无框力矩电机、IMU、深度相机国产化推动 BOM 下降 30–60%；
3. **数据飞轮初现**：Open X-Embodiment、DROID、AgiBot World 等跨本体数据集让"机器人 ImageNet 时刻"逐步显现；
4. **算力支持**：边缘端 Jetson Thor / Orin、车载级算力进入 100–1000 TOPS 区间，可承载 VLA 推理；
5. **政策催化**：中、美、欧均把"人形机器人 / 具身智能"列为战略新兴产业。

## 1.5 本报告关注的边界

- **包含**：人形 / 四足 / 移动操作 / 灵巧手 / 家用与商用服务机器人，以及驱动它们的算法、模型、数据、仿真与产业链；
- **不含 / 弱包含**：纯工业机械臂（除非用于具身研究）、AGV/AMR 物流车、自动驾驶（视为独立赛道）、纯虚拟智能体（agent in browser）。

> 自动驾驶严格说也是具身智能的一种，但产业链和监管路径已经成熟独立，本报告原则上不展开。
