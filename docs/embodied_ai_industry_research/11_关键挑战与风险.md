# 11 关键挑战与风险

## 11.1 技术挑战

### 11.1.1 数据稀缺
- 相比互联网文本（万亿 token）、图像（百亿张），机器人轨迹数据仅在百万量级；
- 真机遥操作成本高（5–50 元/条），仿真存在 sim2real gap，人类视频缺动作标签；
- **没有任何一类数据源能单独支撑 AGI 级具身**——必须组合，但融合方法尚未定型。

### 11.1.2 长尾鲁棒性
- 实验室成功率 90% 与真实现场 99.9% 之间是 **两个数量级的工程差距**；
- 长尾场景：光照、遮挡、变形物体、突发干扰、物体新颖性；
- 与自动驾驶相同的"corner case curse"。

### 11.1.3 实时性与算力
- 端到端 VLA 推理频率多在 5–30 Hz，难以匹配 100–500 Hz 的反应控制；
- 需 **Action Chunking + 双系统** 缓解；
- 边缘 SoC 内存、带宽、功耗共同限制模型规模 ≤ 7 B 参数级别。

### 11.1.4 接触动力学与柔性物体
- 衣物 / 食材 / 软包装 / 流体的建模与仿真仍然原始；
- GPU 物理引擎在 contact-rich 任务上误差大；
- 触觉反馈尚未标准化。

### 11.1.5 跨本体迁移
- 不同机器人尺寸、自由度、传感配置不同，难以共享策略；
- 当前最佳方案（OpenVLA / π0）仍是"训练时见过该本体"；
- 真正的"零样本新本体" 还远。

### 11.1.6 长程任务
- 一个家务任务可能涉及几十步、跨几分钟；
- LLM 规划层 + VLA 执行层之间错误累积严重；
- 失败恢复（recovery）能力差。

### 11.1.7 安全与认证
- 机器人靠近人时需保证 SIL2/3 级别安全；
- 端到端神经网络难以做形式化验证；
- 未来"安全包络 + 神经策略"混合架构是工程必经之路。

## 11.2 工程化挑战

| 维度 | 难点 |
|------|------|
| 量产 | 关节一致性、装配精度、QC 合格率仍低 |
| 维护 | MTBF 几百小时 vs 工业要求几万小时 |
| 续航 | 1–4 小时无法满足一个工班 |
| OTA | 模型 GB 级 OTA 与连接性约束 |
| 调试 | 现场部署一台往往数周 |
| 远程运维 | 多机协同 + 故障诊断尚处早期 |

## 11.3 商业挑战

1. **PMF 模糊**：除了"科研 + 数据采集 + 工业试点"，真正可规模付费的场景仍少；
2. **客户心智**：B 端客户对"演示视频" 与"真机交付"差距警惕；
3. **价格悖论**：售价高 → 客户不买；售价低 → 厂家亏损；唯有量产规模才能解决；
4. **服务交付**：单台部署调试人天数高，毛利受压；
5. **同质化**：中国 30+ 家人形整机厂硬件参数高度趋同。

## 11.4 数据与隐私

- 家庭场景：摄像头、麦克风采集涉及最严重隐私；
- 工业场景：客户产线工艺机密被记录；
- 医疗：HIPAA / NMPA 极严；
- 解决路径：本地推理 + 联邦学习 + 差分隐私 + 数据红线协议。

## 11.5 伦理与社会

| 议题 | 描述 |
|------|------|
| **就业冲击** | 如果替代 1 亿蓝领岗位，将引发劳动力市场剧烈调整 |
| **依赖与孤独** | 陪伴机器人对老人 / 儿童的心理影响仍是开放命题 |
| **责任归属** | 机器人致害时，事故责任如何分配 |
| **军事化** | 四足 / 人形载武器引发的国际人道法问题 |
| **公平性** | 大模型偏见 + 物理身体导致更大现实伤害 |

## 11.6 地缘政治与供应链

- 美国出口管制：高端 GPU、特定关节零件、先进仿真软件；
- 中国关键稀土、原材料反制工具；
- 台海局势对全球先进芯片供应链影响；
- 欧盟单独立法 + 高合规壁垒；
- 数据出境与跨境合规。

## 11.7 资本与商业泡沫

- 2024–2025 年估值飙升明显领先实际营收；
- 部分公司 DAU = "演示量" 而非真实订单；
- 二级市场（A 股、港股 18C、美股）若情绪反转可能引发寒冬；
- 2027 年前后是关键的 "兑现 vs 退潮" 节点。

## 11.9 应对建议

对**整机厂**：
- 早期聚焦 1–2 个高价值垂直场景，避免万能化；
- 自研关键零部件 + 关键模型，建立护城河；
- 数据闭环优先于硬件参数堆叠。

对**模型公司**：
- 与多家整机厂共建数据池；
- 拥抱开源以建立生态；
- 与平台公司（NVIDIA / 云厂）合作降低算力门槛。

对**零部件厂**：
- 优先攻克高利润关键件（丝杠、力矩电机、触觉）；
- 与整机厂"绑定+定制"建立长期供应；
- 借汽车产业链规模化降本。

对**投资人**：
- 拒绝同质化"我也是人形" 投资逻辑；
- 关注**数据闭环**和**真实订单**；
- 重视上游零部件冠军的确定性收益。

## 11.10 小结

具身智能不会是一条直线增长曲线，更像 **"高估值早期 → 量产泡沫 → 短期低谷 → 真正爆发"** 的 Gartner 曲线。理性参与者需要在保持长期信念的同时，**用 PMF、数据、利润这三条标尺持续校准**。
