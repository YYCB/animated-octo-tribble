# 07 应用场景

## 7.1 应用场景全景

```
落地难度（高 → 低）          代表场景
家庭通用 ★★★★★               叠衣、做饭、照护老人
精细服务 ★★★★                 餐厅传菜+整理、零售上货
通用工厂 ★★★★                 多任务装配、跨工位
特种危险 ★★★                  消防、核电、矿山、巡检
固定工业 ★★                   汽车 / 3C 上下料、装配
```

## 7.2 工业制造（短期最大商业市场）

### 典型应用

| 场景 | 任务 | 状态 |
|------|------|------|
| 汽车总装 | 物料转运、内饰安装、扭螺丝、贴胶 | 试点（特斯拉、宝马、奔驰、奇瑞、蔚来） |
| 3C 电子 | 焊接、贴片、组装、检测 | 试点（富士康、立讯） |
| 新能源电池 | 模组转运、装配 | 试点（宁德、比亚迪） |
| 物流 / 仓储 | 拣货、码垛、上下架 | 部分量产（Agility Digit、智元） |
| 机加工 | 上下料、装夹 | 量产（机械臂为主） |

### 经济性测算（参考）

```
单台 25 万元 / 寿命 5 年 / 残值 0
人形机器人单工时综合成本 ≈ 25 / (5×250工×2班×8h) ≈ 12.5 元/小时
+ 维护 + 运营 ≈ 18–25 元/小时
中国一线工厂工人综合成本 ≈ 30–50 元/小时
理论 1.5–2 年回本，但目前可靠性不足，实际仍需补贴
```

要点：人形机器人的工业落地，**关键不是"能做"，而是"24×7 不出错"**。当前 MTBF 与节拍仍是核心瓶颈。

## 7.3 商业服务

| 子场景 | 形态 | 案例 |
|--------|------|------|
| 餐饮 | 移动操作 / 双臂 | 普渡、擎朗送餐、稻草人咖啡机器人 |
| 零售 | 上货 / 盘点 | Simbe、智元零售试点 |
| 酒店 | 配送、清洁、迎宾 | 云迹、擎朗 |
| 医院 | 配送药品、消毒、康复 | UVD、傅利叶康复机器人 |
| 商场 / 政务大厅 | 导览、表演 | 多家 |
| 银行 / 营业厅 | 大堂引导 | 试点 |

特点：环境结构化但与人共存，**安全合规重要性 ≥ 性能**。

## 7.4 物流与仓储

- 已经成为 AMR / AGV 的舒适区，人形或双臂主要解决 **"最后一米"**：拆码垛、整理货架、补货；
- Agility Digit 已与 GXO、Amazon 进入实际试点；
- 国内顺丰、京东、菜鸟、美团均有试点。

## 7.5 家庭场景（远期 / 想象空间最大）

任务难度极不均匀：
- **较易**：清洁、垃圾分类、自主巡检；
- **中**：取放物品、整理桌面、衣物分拣；
- **难**：叠衣（柔性物体）、做饭（多步骤 + 高温）、照护老人；
- **极难**：开放对话陪伴、儿童照护、宠物互动。

挑战：
- 长尾场景多、安全要求最高、噪音 / 续航 / 外观 / 成本约束极强；
- 价格目标：进入家庭需 ≤ 10 万元（早期发烧友），普及需 ≤ 3 万元（5–10 年）；
- 监管路径仍未清晰。

## 7.6 特种应用

| 子场景 | 形态偏好 | 现状 |
|--------|---------|------|
| 电力巡检 | 四足 / 人形 | 国家电网批量采购四足 |
| 石化 / 燃气 | 四足 + 防爆 | 试点 |
| 消防救援 | 四足 / 履带 | 试点（防火、侦察） |
| 矿山 / 隧道 | 四足 / 履带 | 试点 |
| 反恐 / 国防 | 四足 + 武器化（敏感） | 各国谨慎布局 |
| 核电 | 抗辐射 | 试点 |

特种应用商业模式以"政府订单 + 国央企集采"为主，单价高但量小。

## 7.7 医疗与康复

| 场景 | 形态 |
|------|------|
| 康复训练（外骨骼/上肢） | 傅利叶、布法罗、Ekso |
| 手术机器人（da Vinci 系） | 不属本报告核心 |
| 老人陪伴 / 监护 | 仍研究阶段 |
| 院内配送 / 消毒 | 已商用 |

医疗具身智能受 FDA / NMPA 严格监管，**节奏慢于其它场景**，但客单价最高。

## 7.8 科研与教育

- 高校 / 研究所大量采购四足、双臂、人形作为研究平台；
- 宇树 G1、AgileX Cobot Magic、Trossen ALOHA 是标配；
- 教育市场（K12、职业技能教育）逐渐起量；
- 数据采集业务："数据工厂" 正成为新职业类别。

## 7.9 落地关键评估维度

| 维度 | 内涵 | 优先级 |
|------|------|-------|
| 可重复性 | 同任务长时间稳定 | ★★★★★ |
| 节拍 | 每小时完成数 | ★★★★★ |
| 安全性 | 与人共存 | ★★★★★ |
| 维护成本 | MTBF / 修复成本 | ★★★★ |
| 部署复杂度 | 现场调试时间 | ★★★★ |
| 数据闭环 | 部署越多越聪明 | ★★★★ |
| 成本下降斜率 | 5 年内降幅 | ★★★ |

## 7.10 场景成熟度雷达图（定性）

```
                              工业（汽车/3C/新能源）
                                   ●
                          ●                ●
                  特种 / 电力巡检         物流仓储
                                            
       ●                                            ●
      家庭                                           商业服务
                                            
                          ●                ●
                       医疗康复          科研教育
                                   
                              （越接近外圈，越接近成熟落地）

   外圈：可商用规模化   中圈：可签单试点   内圈：仍在 PoC
```

实际值（粗略）：
- **外圈**：科研教育、商业服务（部分子场景）、四足特种；
- **中圈**：工业制造、物流仓储、电力巡检；
- **内圈**：通用家庭、长程精细任务、跨场景人形。

## 7.11 小结

落地节奏判断：
- **2024–2026**：科研、特种、轻工业；
- **2026–2028**：汽车 / 3C 工厂、商业服务规模化、物流"最后一米"；
- **2028–2030**：B 端通用工人替代试点放量；
- **2030+**：家庭 / C 端开始进入早期采用者。
