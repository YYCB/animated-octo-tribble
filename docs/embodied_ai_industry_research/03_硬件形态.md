# 03 硬件形态

## 3.1 形态全景

```
具身智能本体形态
├── 人形机器人 Humanoid             —— 全场景兼容、最高复杂度
│   ├── 全尺寸（150–180 cm）
│   └── 半尺寸 / 桌面形（< 1 m）
├── 双足 / 仿生            —— 介于人形与四足之间
├── 四足机器人 Quadruped           —— 户外巡检、教育科研、负载搬运
├── 轮式 / 履带 + 上身             —— 室内服务、工厂搬运（性价比高）
│   └── 轮式底盘 + 双臂操作
├── 单臂 / 双臂操作器              —— 桌面操作、零售、料理
├── 灵巧手 / 末端执行器             —— 关键独立赛道
├── 移动作业机器人 Mobile Manipulator
└── 软体 / 仿生 / 微型             —— 学术与特种应用
```

## 3.2 各形态对比

| 形态 | 自由度 | 典型成本 | 主要优势 | 主要瓶颈 | 代表产品 |
|------|--------|---------|---------|---------|---------|
| 全尺寸人形 | 25–55 | 5–80 万元 | 场景通用、人类工具兼容、想象空间大 | 平衡控制难、能耗高、安全风险高 | Tesla Optimus、Figure 02/03、Apptronik Apollo、宇树 H1/G1、智元远征 A2、Fourier GR-1/2、星动纪元 XBot、1X NEO |
| 双足非人形 | 12–20 | 中 | 结构简化 | 上半身能力受限 | Agility Digit |
| 四足 | 12–20 | 1–10 万元 | 越野能力强、技术成熟、量产可行 | 没有手、负载有限 | 波士顿动力 Spot、ANYmal、宇树 Go2/B2、云深处绝影、蔚蓝 Alpha Dog |
| 轮式 + 双臂 | 14–20 | 中低 | 室内成熟、续航好、稳定 | 越障能力差 | Stretch (Hello Robot)、波士顿动力 Stretch、Mobile-ALOHA、追觅 / 星尘 / 优艾智合等 |
| 桌面双臂 | 12–14 | 低 | 数据采集和研究友好 | 无移动 | ALOHA、Trossen WidowX、AgileX Cobot Magic |
| 灵巧手 | 6–22 | 数千–数十万 | 是人形可用性的关键 | 寿命、力度、成本 | Shadow、Inspire、PaXini、星动 XHand、特斯拉手 |
| 软体 / 仿生 | — | 学术 | 安全、灵活 | 控制建模困难 | 章鱼臂、气动手 |

## 3.3 人形机器人深入剖析

### 3.3.1 关键指标

| 指标 | 当前主流水平（2025–2026 量产级） | 目标 |
|------|-------------------------------|------|
| 身高 | 1.2–1.8 m | 与人匹配 |
| 体重 | 30–80 kg | < 60 kg 更安全 |
| 自由度 | 25–45 | 50+（含手） |
| 续航 | 1–4 小时 | 8 小时（一个工班） |
| 步速 | 1–2 m/s | 1.5–2 m/s 持续 |
| 负载 | 5–20 kg | 25 kg+ |
| 单关节峰值扭矩 | 50–360 Nm | 视位置而定 |
| BOM 成本 | 10–50 万元 | < 10 万元（消费级） |
| MTBF | 数百小时 | 数万小时 |

### 3.3.2 关键子系统国产化进度

| 子系统 | 国产化情况 | 主要厂家 |
|--------|-----------|----------|

| 行星滚柱丝杠 | 中（2024 起放量） | 恒立液压、北特科技、贝斯特 |
| 无框力矩电机 | 中（高端仍依赖进口） | 步科、鸣志、雷赛 |
| 关节模组 | 高 | 宇树自研、智元、本末科技、兆威机电 |
| 6 轴力 / 力矩传感器 | 中 | 坤维、宇立仪器、鑫精诚 |
| 触觉传感 | 早期 | 帕西尼、坤维、戴盟、他山 |
| IMU | 中 | 矽睿、芯动联科、北微 |
| 深度相机 | 高 | 奥比中光、图漾、华捷艾米 |
| 主控算力 | 中 | NVIDIA Jetson 仍主流，地平线 / 寒武纪 / 黑芝麻在追赶 |

### 3.3.3 设计哲学分歧

1. **"工业实用派"**：Apptronik、Agility、Figure 02——优先工厂落地，强调可靠性、安全围栏、节拍。
2. **"通用家用派"**：1X NEO、Tesla Optimus（长期）——聚焦家庭与陪伴，强调外观、安全、低噪音。
3. **"高性能演示派"**：Boston Dynamics Atlas（电动版）、Unitree H2——突破极限运动学，研究价值高。
4. **"开源平民派"**：Berkeley Humanoid Lite、HumanPlus、Open-TeleVision、宇树 G1（部分开放）——降低研究门槛。

## 3.4 灵巧手单独成赛道

灵巧手已经成为人形机器人最关键的差异化点。设计维度：

| 维度 | 选项 |
|------|------|
| DOF | 6 / 11 / 16 / 22 |
| 驱动 | 腱驱、连杆驱、直驱、气动、电机内置 |
| 触觉 | 无 / 指尖压力 / 光学（GelSight）/ 大面积电容阵列 |
| 寿命 | 数百小时（Shadow 旧）→ 数千小时（新一代）|
| 成本 | 数千元（教学）→ 30 万元（科研旗舰） |

代表玩家：
- 海外：Shadow Robot、HaptX、Sanctuary AI、Tesla Hand（自研）
- 国产：因时机器人、Inspire（睿尔曼）、星动 XHand、帕西尼 PaXini DexH13、银河通用配套手、Linker Hand、知行机器人

**关键技术趋势**：
1. 全驱（fully-actuated）→ 欠驱动（under-actuated）的回归，以降低成本；
2. 触觉成为标配，"看 + 摸"双模态闭环；
3. 模块化、易换，故障件可现场更换。

## 3.5 移动底盘

具身机器人的"腿"另一种主流是轮式 / 履带：
- 差速轮、麦克纳姆轮、阿克曼、四轮全向；
- 性价比高，室内场景成熟；
- 适合零售、餐厅、医院巡检；
- 与上身机械臂集成形成 **Mobile Manipulator**，是工业落地最快的具身形态之一。

> 本仓库 `chassis_protocol/` 模块即面向轮式底盘的 ROS2 HAL，对应这一类"轮式 + 上身"产品线的工程实现。

## 3.6 计算与电源

- **算力分布**：通常采用"主控 SoC（Jetson Orin/Thor 或 x86）+ 实时控制 MCU（STM32 / TI / 自研）+ 关节级驱动 IC"的三级架构；
- **网络**：EtherCAT / CAN-FD / 千兆以太网 / 自研 RS485 三层并存；
- **电源**：21–58 V 锂电池为主流，能量密度 200–280 Wh/kg；
- **散热**：高负荷下关节温升与积热是稳定性核心瓶颈，被动散热 + 强制风冷较常见。

## 3.7 形态选择决策

```
是否需要使用人类工具 / 走人类楼梯？
├── 是 ──► 需要人形 or 高 DOF 双臂
│         ├── 强机动性 ──► 人形
│         └── 室内为主 ──► 轮式 + 双臂（更省钱）
└── 否 ──►
   ├── 户外巡检 ──► 四足
   ├── 桌面研究 ──► 双臂示教器（ALOHA 类）
   └── 工业搬运 ──► AMR / 工业机械臂（不在本报告核心）
```

短期投资性价比上，**轮式 + 双臂的 Mobile Manipulator** 仍是最现实的"先跑通"形态；人形是中长期想象空间最大的押注。
