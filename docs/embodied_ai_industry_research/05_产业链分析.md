## 5.2 上游：核心零部件

### 5.2.1 BOM 结构（人形机器人参考）

| 子系统 | 占 BOM 比例（参考） | 主要供应方 |
|--------|---------------------|------------|
| 关节模组 | 35–45% | 自研 + 谐波 / 行星 / 电机厂 |
| 灵巧手 | 8–15% | 自研 / 因时 / Inspire / PaXini |
| 传感器（视觉+IMU+力+触觉） | 8–15% | 奥比中光 / Realsense / 矽睿 / 帕西尼 |
| 计算 + 主板 | 5–10% | NVIDIA / 地平线 / 英特尔 |

| 结构件（铝合金 / 碳纤维 / 镁合金） | 8–12% | 立讯 / 拓普 / 长盈 |

> 比例为业界经验区间，单机售价 10–80 万元区间内有显著差异，规模化后关节占比会下降，模型 / 软件成本占比上升。

### 5.2.3 上游关键趋势

1. **垂直整合**：头部本体厂（特斯拉、Figure、智元、宇树）大幅自研关节、电机甚至灵巧手；
2. **汽车供应链溢出**：丝杠、电机、激光雷达、IMU 借汽车产业链快速量产降本；
3. **触觉成为新蓝海**：尚无标准、无寡头，是 2025–2027 年潜在弯道超车窗口；
4. **国产替代红利**：减速器、电机、传感器等核心件国产化率快速提升，价格 1/3 ~ 1/2 海外。

## 5.3 中游：本体与基础模型

### 5.3.1 商业模式分类

| 模式 | 描述 | 代表 |
|------|------|------|
| **整机厂（One-Stack）** | 同时做硬件 + 软件 + 大模型 | Tesla、Figure、Apptronik、智元、宇树、星动纪元、银河通用 |
| **基础模型公司** | 不做硬件，做"机器人 GPT"卖能力 | Physical Intelligence (π)、Skild AI、Covariant（已被 Amazon 收编核心团队） |
| **平台 / 中间件** | 提供 OS / 仿真 / 工具链 | NVIDIA（Isaac、GR00T、Cosmos）、华为、英特尔 |
| **零部件 + 模组** | 卖关节、灵巧手 | 绿的谐波、因时、Inspire、PaXini、本末科技 |
| **应用集成 / 运营** | 把机器人卖给场景 / 收服务费 | 多数家电、零售、物流公司 |

### 5.3.2 整机厂的两条路径

```
路径 A: 量产先行    路径 B: 模型先行
路径 A：硬件 → 量产 → 数据飞轮 → 模型自然涌现
   代表：Tesla、宇树、智元、Figure
   赌点：先抢硬件市场入口，模型差距日后追赶

路径 B：先做"通用大脑"，硬件次要
   代表：Physical Intelligence、Skild、谷歌 DeepMind
   赌点：跨本体模型一旦突破，本体成商品化外设
```

二者最终大概率会在某一时点交汇——这也是当前竞争最激烈的战场。

## 5.4 下游：场景生态

下游集成方决定了**最终单台机器人 ROI**。常见集成形态：
- **直销 + 自运营**：自有客户、自维护，毛利高但起量慢；
- **总包集成**：与传统系统集成商（SI）合作，进入工厂、医院、酒店；
- **租赁 / RaaS**：按月或按工时收费，降低客户购置门槛；
- **数据回流**：场景方反向贡献训练数据。

下游详细场景见第 07 章。

## 5.6 国产产业链优势与劣势

**优势**：
- 完整制造业基础（汽车、3C、家电外溢）；
- 减速器 / 电机 / 传感器国产化率上升迅速；
- 大模型与开源能力相对靠前；
- 政策与资本双重支持；
- 应用场景密集（制造业 + 服务业体量全球第一）。

**劣势**：
- 高端芯片（边缘大算力）受限；
- 高端力 / 力矩 / 触觉传感精度与寿命仍弱；
- 操作系统与中间件长期依赖 Linux 生态；
- 顶级 VLA 基础模型仍以海外为主，开源也是 Meta / Google / Stanford 主导；
- 标准与认证体系尚未建立。

## 5.7 小结

- 上游：**减速器 / 丝杠 / 电机国产化** 已经发生，**触觉与高端传感** 是下一轮机会；
- 中游：**整机厂百花齐放，2026–2028 年开始洗牌**；具身基础模型公司是少数几家全球级别；
- 下游：**先工业 / 商业场景，再家庭**；具备场景闭环和数据回流能力的玩家最有壁垒；
- 长期价值将向 **数据 + 模型 + 软件** 集中。
