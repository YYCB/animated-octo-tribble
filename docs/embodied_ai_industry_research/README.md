## 一句话结论

**具身智能 = 大模型时代的"身体"**，是 2023 年以来 AI 产业最具确定性的下一波浪潮之一：
技术上正经历从"模块化 pipeline"向"VLA / 世界模型驱动的端到端"范式迁移；
产业上呈现"硬件本体百花齐放、数据稀缺成核心瓶颈、垂直场景先于通用家庭落地"的格局；
资本与政策双轮驱动下，2024–2026 年是关键窗口期，但**真正实现"通用、可靠、可负担"** 的家用机器人仍需 5–10 年。
