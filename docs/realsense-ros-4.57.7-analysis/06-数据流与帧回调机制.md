## 2. 帧回调入口函数

### 2.1 视频帧回调 (`frame_callback_function`)

```cpp
// 定义在 rs_node_setup.cpp → setAvailableSensors()
std::function<void(rs2::frame)> frame_callback_function = [this](rs2::frame frame) {
    bool is_filter = any_of(_filters, [](f) { return f->is_enabled(); });
    
    if (_sync_frames || is_filter) {
        // 通过异步同步器处理
        // 同步器会将多个流的帧合并为 frameset
        this->_asyncer.invoke(frame);
    } else {
        // 直接处理单帧
        frame_callback(frame);
    }
};
```

### 2.2 IMU 回调 (`imu_callback_function`)

```cpp
std::function<void(rs2::frame)> imu_callback_function = [this](rs2::frame frame) {
    imu_callback(frame);          // 独立发布 accel/gyro/motion
    if (_imu_sync_method != NONE)
        imu_callback_sync(frame); // 融合发布 united IMU
};
```

### 2.3 异步同步器设置

```cpp
void BaseRealSenseNode::setCallbackFunctions() {
    _asyncer.start([this](rs2::frame f) {
        frame_callback(f);  // 同步器输出到 frame_callback
    });
}
```

`rs2::asynchronous_syncer` 是 librealsense2 提供的帧同步器：
- 输入：各传感器的独立帧
- 输出：时间对齐的 frameset（包含所有流的帧）

## 3. 核心帧处理 (`frame_callback()`)

```cpp
void BaseRealSenseNode::frame_callback(rs2::frame frame) {
    // ---- 时间基准初始化 ----
    double frame_time = frame.get_timestamp();
    if (!_is_initialized_time_base) {
        _is_initialized_time_base = setBaseTime(frame_time, time_domain);
    }
    
    // ---- Frameset 处理 ----
    if (frame.is<rs2::frameset>()) {
        auto frameset = frame.as<rs2::frameset>();
        
        // 暂停 IMU 发布（避免 IMU 与图像时间错乱）
        if (_synced_imu_publisher) _synced_imu_publisher->Pause();
        
        // 应用滤波器管道
        for (auto& filter : _filters) {
            if (filter->is_enabled())
                frameset = filter->process(frameset);
        }
        
        // 遍历每一帧
        for (auto it = frameset.begin(); it != frameset.end(); ++it) {
            auto f = *it;
            auto sip = {f.stream_type(), f.stream_index()};
            auto t = frameSystemTimeSec(f);
            
            // 按流类型分发
            switch(stream_type) {
                case RS2_STREAM_LABELED_POINT_CLOUD:
                    publishLabeledPointCloud(f.as<labeled_points>(), t);
                    break;
                case RS2_STREAM_OCCUPANCY:
                    publishOccupancyFrame(f, t);
                    break;
                default:
                    publishFrame(f, t, sip, _images, _info_publishers, _image_publishers);
                    
                    // 对齐深度发布
                    if (align_depth_enabled && sip == DEPTH) {
                        publishFrame(f, t, sip, ..., _depth_aligned_image_publishers);
                    }
                    
                    // RGBD 组合发布
                    if (rgb/depth ready) publishRGBD(...);
            }
        }
        
        // 恢复 IMU 发布
        if (_synced_imu_publisher) _synced_imu_publisher->Resume();
    }
    // ---- 单帧处理 ----
    else {
        auto sip = {frame.stream_type(), frame.stream_index()};
        auto t = frameSystemTimeSec(frame);
        publishFrame(frame, t, sip, ...);
    }
}
```

## 4. 时间同步机制

### 4.1 时间基准设置

```cpp
bool BaseRealSenseNode::setBaseTime(double frame_time, rs2_timestamp_domain time_domain) {
    if (time_domain == RS2_TIMESTAMP_DOMAIN_SYSTEM_TIME) {
        // 硬件时间戳基于系统时钟
        // _camera_time_base = 0（不需要偏移）
        return true;
    }
    // 其他情况：计算相机时钟与系统时钟的偏移
    _camera_time_base = frame_time;
    _ros_time_base = _node.now();
    return true;
}
```

### 4.2 帧时间转换

```cpp
rclcpp::Time BaseRealSenseNode::frameSystemTimeSec(rs2::frame frame) {
    double elapsed_camera_ns;
    
    if (frame.get_frame_timestamp_domain() == RS2_TIMESTAMP_DOMAIN_SYSTEM_TIME) {
        // 直接使用系统时间（毫秒→纳秒）
        return rclcpp::Time(millisecondsToNanoseconds(frame.get_timestamp()));
    }
    
    // 相机时钟：需要加上偏移
    elapsed_camera_ns = millisecondsToNanoseconds(frame.get_timestamp() - _camera_time_base);
    return rclcpp::Time(_ros_time_base.nanoseconds() + elapsed_camera_ns);
}
```

## 5. 帧发布详情 (`publishFrame()`)

```cpp
void BaseRealSenseNode::publishFrame(
    rs2::frame f,
    const rclcpp::Time& t,
    const stream_index_pair& stream,
    map<sip, cv::Mat>& images,
    const map<sip, CameraInfoPub>& info_publishers,
    const map<sip, ImagePub>& image_publishers,
    bool is_publishMetadata = true)
{
    // 1. 检查是否有订阅者
    unsigned int width = video_frame.get_width();
    unsigned int height = video_frame.get_height();
    
    // 2. 填充 CV Mat
    fillCVMatImageAndReturnStatus(f, images, width, height, stream);
    
    // 3. 深度帧特殊处理
    if (stream == DEPTH) {
        fix_depth_scale(images[stream], images[stream]);  // 标准化深度单位
        if (_clipping_distance > 0)
            clip_depth(depth_frame, _clipping_distance);  // 裁切远处
    }
    
    // 4. 填充 ROS Image 消息
    sensor_msgs::msg::Image img_msg;
    fillROSImageMsgAndReturnStatus(images[stream], stream, width, height, 
                                    stream_format, t, &img_msg);
    
    // 5. 发布 Image
    if (image_publishers.count(stream) && subscribers > 0) {
        image_publishers[stream]->publish(img_msg);
    }
    
    // 6. 发布 CameraInfo
    if (info_publishers.count(stream) && subscribers > 0) {
        auto info_msg = _camera_info[stream];
        info_msg.header.stamp = t;
        info_publishers[stream]->publish(info_msg);
    }
    
    // 7. 发布 Metadata
    if (is_publishMetadata) {
        publishMetadata(f, t, frame_id);
    }
}
```

## 6. RGBD 组合发布

```cpp
void BaseRealSenseNode::publishRGBD(
    const cv::Mat& rgb_cv_matrix,
    const rs2_format& color_format,
    const cv::Mat& depth_cv_matrix,
    const rs2_format& depth_format,
    const rclcpp::Time& t)
{
    // 条件检查
    if (!_rgbd_publisher || _rgbd_publisher->get_subscription_count() == 0) return;
    
    auto msg = RGBD();
    msg.header.stamp = t;
    msg.header.frame_id = OPTICAL_FRAME_ID(COLOR);
    
    // 填充 RGB 图像
    fillROSImageMsgAndReturnStatus(rgb_cv_matrix, COLOR, ..., &msg.rgb);
    
    // 填充 Depth 图像（对齐到 Color 的深度）
    fillROSImageMsgAndReturnStatus(depth_cv_matrix, DEPTH, ..., &msg.depth);
    
    // 填充 CameraInfo
    msg.rgb_camera_info = _camera_info[COLOR];
    msg.depth_camera_info = _camera_info[DEPTH];
    
    _rgbd_publisher->publish(msg);
}
```

**前置条件**：RGBD 话题需要同时满足：
1. `enable_rgbd = true`
2. `sync_frames = true`（帧同步）
3. Color 流已启用
4. Depth 流已启用
5. `align_depth` 滤波器已启用

## 8. SyncedImuPublisher

用于管理 IMU 消息与图像帧的时序关系：

```
                  frame_callback
                       │
         ┌─────────────┴─────────────┐
         │                           │
    Pause()                      Resume()
         │                           │
    ┌────▼────┐                 ┌────▼────┐
    │ 缓存 IMU │  ──→──→──→──  │ 发布缓存 │
    │ 消息队列  │  (处理帧期间)   │ 中的 IMU │
    └─────────┘                └─────────┘
```

```cpp
class SyncedImuPublisher {
    void Pause() {  _pause_mode = true; }     // 开始缓存
    void Resume() { PublishPendingMessages(); _pause_mode = false; }  // 释放缓存
    void Publish(imu_msg) {
        if (_pause_mode)
            _pending_messages.push(imu_msg);  // 暂存
        else
            _publisher->publish(imu_msg);     // 直接发布
    }
};
```

## 9. 对 HAL 设计的启示

### 9.1 数据管道接口

```
IFramePipeline:
  + registerCallback(stream_type, callback)
  + enableSync(enabled: bool)
  + addFilter(filter: IFilter)
  + removeFilter(filter: IFilter)
  + start()
  + stop()

IFrameCallback:
  + onFrame(frame: Frame)
  + onFrameSet(frameset: FrameSet)

Frame:
  + getStreamType() → StreamType
  + getTimestamp() → double
  + getTimestampDomain() → TimestampDomain
  + getData() → void*
  + getWidth() / getHeight() (for video)
  + getFormat() → Format
```

### 9.2 关键抽象点
1. **帧同步**：SDK 提供的 `asynchronous_syncer` 是关键组件，HAL 需要抽象
2. **时间同步**：系统时间 vs 硬件时间的转换逻辑需要通用化
3. **IMU 缓存机制**：`SyncedImuPublisher` 的暂停/恢复模式可以泛化
4. **按需发布**：只在有订阅者时才处理和发布数据（节省 CPU）
